#### 安装教程

1. 仅支持Centos7以上的系统
2. 登录系统执行 `wget --no-check-certificate https://gitee.com/yiangzhao/openvpn/raw/master/fas.sh && chmod +x  fas.sh && bash fas.sh`
3. 按照提示进行安装

#### 使用说明

1. 内存最低512M 硬盘至少200M
2. 系统为Centos7.0或更高
